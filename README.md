**📌 OptiCode – AI-Powered Code Reviewer**

🚀 OptiCode is a smart code review tool that helps developers analyze and improve their code efficiently. It provides automated feedback, detects code quality issues, and ensures best practices in programming.


**🛠 Features**

✅ Automatic Code Review – Get instant feedback on your code

✅ Syntax & Style Checking – Detect syntax errors and formatting issues

✅ Performance Analysis – Identify performance bottlenecks

✅ Security Vulnerability Detection – Highlight potential security risks

✅ Supports Multiple Languages – Review code in various programming languages


**⚙️ Tech Stack**

🔹 Frontend: React, Tailwind CSS

🔹 Backend: Node.js, Express.js

🔹 Database: MongoDB

🔹 AI/Analysis: Gemini API


**🚀 Getting Started**

1️⃣ Clone the Repository

    https://github.com/SakshiSinglaa/OptiCode-AI-based-Code-Reviewer.git
    
    cd OptiCode

2️⃣ Install Dependencies

  **Backend**
  
    cd server
    
    npm install
  
  **Frontend**
  
    cd client
    
    npm install
    
3️⃣ Set Up Environment Variables
    
    Create a .env file in the root directory and add:
    
      MONGO_URI=your_mongodb_connection_string
      
      API_KEY=your_api_key
      
      PORT=5000
      
4️⃣ Run the Project
  
  **Backend**
  
    cd server
    
    npm start
  
  **Frontend**
  
    cd client
    
    npm start
